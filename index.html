Custom Linux
MiniShell

A lightweight custom shell implemented in C that mimics core Linux terminal behavior including command execution, built-in commands, and process management.

Source Code Watch Demo

Why MiniShell?

Command Execution

Supports execution of external Linux commands using fork and exec.

Built-in Commands

Implements internal commands like cd, exit and environment handling.

Process Handling

Manages child processes using wait and system-level calls.

Key Features

  • • Execution of external commands (e.g., ls, wc, sleep)
  • • Built-in internal commands such as cd, pwd, and exit
  • • Command-line input parsing and argument handling
  • • Process control using fork(), execvp(), and waitpid()
  • • Signal handling using SIGINT and SIGCHLD
  • • Stable shell behavior with proper child process management

Technology Stack

C Programming Linux Internals File I/O (fopen, fread, fwrite) POSIX System Calls Process Control (fork, exec, wait) Signal Handling Modular Programming
