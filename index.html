Custom Linux
MiniShell

A lightweight custom shell implemented in C that mimics core Linux terminal behavior including command execution, built-in commands, and process management.

Source Code Watch Demo

Why MiniShell?

Command Execution

Supports execution of external Linux commands using fork and exec.

Built-in Commands

Implements internal commands like cd, exit and environment handling.

Process Handling

Manages child processes using wait and system-level calls.

Key Features

  • Command parsing and tokenization
  • Fork and exec system call usage
  • Environment variable handling
  • Error handling and input validation
  • Modular source code structure
